# OG 图片放这里

把你生成好的结果图直接放进当前文件夹，并使用下面这些固定文件名：

- `system-operator.png`
- `hidden-villain.png`
- `performative-savior.png`
- `cold-survivor.png`
- `higher-observer.png`
- `emotion-parasite.png`
- `rule-breaker.png`
- `silent-hunter.png`
- `chosen-substitute.png`
- `unstable-variable.png`

如果你还要放一个默认图，文件名建议用：

- `og-default.png`

这些图片放好后，前端会按 `src/data/identities.ts` 里的 `ogImage` 字段取图。
